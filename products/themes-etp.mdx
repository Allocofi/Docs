Each ETP unit represents a proportional ownership interest in the underlying assets. The fair value of an ETP is determined through a transparent NAV calculation.

The fundamental pricing relationship is:

$$
\boxed{
ETP\ Price_t=\frac{NAV_t}{Units_t}
}
$$

## Key Components

| Component | Description |
| --- | --- |
| NAV | Total net value of the ETP portfolio |
| Units | Total outstanding ETP supply |
| ETP Price | Fair value per ETP unit |

# 1. Underlying Portfolio Valuation

An ETP may hold multiple assets including equities, ETFs, commodities, or other financial instruments.

The value of each position is calculated based on:

$$
Position\ Value_i=Quantity_i \times Market\ Price_i
$$

The total portfolio value is:

$$
Portfolio\ Value_t=
\sum_{i=1}^{n}
(Q_i \times P_{i,t})
$$

# 2. Net Asset Value (NAV)

NAV represents the total economic value backing the ETP.

The NAV calculation includes assets, cash balances, income, fees and liabilities.

$$
NAV_t=
Assets_t+
Cash_t+
Income_t-
Fees_t-
Liabilities_t
$$

Where:

| Component | Description |
| --- | --- |
| Assets | Market value of underlying holdings |
| Cash | Cash or stablecoin balance |
| Income | Accrued income generated by assets |
| Fees | Management and operational expenses |
| Liabilities | Outstanding obligations |

# 3. ETP Unit Pricing

The ETP unit price is calculated by dividing NAV by the total outstanding units.

$$
\boxed{
Price_t=
\frac{NAV_t}{Supply_t}
}
$$

Example:

An ETP has:

- NAV: \$10,000,000
- Outstanding units: 1,000,000

The unit price is:

$$
\frac{10,000,000}{1,000,000}=10
$$

Therefore:

$$
\boxed{
ETP\ Price=\$10
}
$$

# 4. Mint Pricing

When a user creates new ETP units, the number of units received is based on the current NAV per unit.

$$
\boxed{
Minted\ Units=
\frac{Deposit-Fee}
{NAV\ per\ Unit}
}
$$

Where:

| Component | Description |
| --- | --- |
| Deposit | Amount deposited by the user |
| Fee | Minting fee charged by the protocol |
| NAV per Unit | Current ETP fair value |

## Example

User deposits:

$$
10,000\ USDC
$$

Mint fee:

$$
0.1\%
$$

Fee amount:

$$
10,000\times0.001=10
$$

Net investment:

$$
9,990
$$

Current NAV per unit:

$$
1.20
$$

Issued units:

$$
\frac{9,990}{1.20}=8,325
$$

The user receives:

$$
\boxed{
8,325\ ETP\ Units
}
$$

# 5. Redemption Pricing

When redeeming ETP units, holders receive the corresponding NAV value.

Gross redemption value:

$$
Redemption\ Value=
Units\times NAV\ per\ Unit
$$

After redemption fees:

$$
USDC_{Received}
=
Units\times NAV_{Unit}
\times(1-Fee)
$$

Where:

| Component | Description |
| --- | --- |
| Units | Amount of ETP tokens redeemed |
| NAV per Unit | Settlement price |
| Fee | Redemption fee rate |

# 6. Portfolio Allocation

Each asset contributes to NAV according to its portfolio weight.

$$
Weight_i=
\frac{Asset\ Value_i}{NAV}
$$

Portfolio weights satisfy:

$$
\sum_{i=1}^{n}Weight_i=1
$$

Where:

| Component | Description |
| --- | --- |
| Asset Value | Market value of each holding |
| Weight | Percentage allocation within the ETP |

# 7. NAV Performance

The return of an ETP is measured by the change in NAV over time.

$$
Return_t=
\frac{NAV_t-NAV_{t-1}}
{NAV_{t-1}}
$$

For a multi-asset portfolio:

$$
Return_{ETP}
=
\sum_{i=1}^{n}
Weight_i
\times
Return_i
$$

---

# 8. Premium and Discount

Secondary market prices may temporarily differ from NAV.

Premium:

$$
Premium=
\frac{Market\ Price-NAV}
{NAV}
$$

Discount:

$$
Discount=
\frac{NAV-Market\ Price}
{NAV}
$$

# Pricing Engine Summary

The complete pricing model:

$$
\boxed{
ETP\ Price
=
\frac{
\sum(Q_iP_i)+Cash+Income-Fees-Liabilities
}
{
Total\ Units
}
}
$$
